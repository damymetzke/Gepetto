(inner) AddSvgObject(name, data)

add a new object to the svg panel.

if the object already exists nothing will happen. to update an existing object use module:SvgManager~UpdateSvgObject.

Parameters:
Name Type Description
name String

unique name of the svg object, the same as the draw object.
data Object

object containing the data of the svg object. object uses the following values:

{
    content: String(), //svg-xml formatted string, this will be directly added to a <g> tag
    transform: new Transform() //(optional) the transform of the object
}

(inner) UpdateSvgObject(name, data)

update an existing object in the svg panel.

if the object does not exist nothing will happen. to add a new object use module:SvgManager~AddSvgObject.

Parameters:
Name Type Description
name String

unique name of the svg object, the same as the draw object.
data Object

object containing the data of the svg object. object can use the following values, but all are optional:

{
    content: String(), //svg-xml formatted string, this will be directly added to a <g> tag
    transform: new Transform() //the transform of the object
}
